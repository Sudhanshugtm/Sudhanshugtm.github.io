# Images Directory

This directory contains all image assets for the portfolio.

## Structure

- **profile/** - Profile photos and headshots
- **projects/** - Project screenshots and case study images
- **icons/** - Icons, logos, and small graphics

## Image Guidelines

- Use WebP format when possible for better compression
- Optimize images before adding them (use tools like Squoosh, ImageOptim, or TinyPNG)
- Keep filenames lowercase with hyphens (e.g., `profile-photo.jpg`)
- Recommended dimensions:
  - Profile photo: 400x400px minimum
  - Project screenshots: 1200px width minimum
  - Icons: 64x64px or SVG

## Adding Images

1. Place images in the appropriate subdirectory
2. Reference them in your HTML/Markdown using: `/assets/images/[subdirectory]/[filename]`
3. Always add descriptive alt text for accessibility
